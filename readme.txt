Semerkand Radyo online yayını dinleme bileşenidir.

== Description ==

Semerkand Radyo Dinle Bieleşeni ile web sitenizin ziyaretçilerine Semerkand Radyo dinlemelerini sağlayabilirsiniz. Daha fazla bilgi edinmek için http://www.jn7.net/semerkand-radyo adresini kullanabilirsiniz. yazyının orjinali http://www.semerkandradyo.com.tr adresinde yapılmaktadır. 

Some features:

    * Bileşen ayarları penceresinden radyo boyutları ayarlanabilir.

    * Bileşen başlığı değiştirilebilir

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
